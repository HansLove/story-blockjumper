## PROBLEMAS ESPECÍFICOS DE ESCRITURA

### 1. Errores gramaticales/ortográficos:

**Línea 4**: `se expresión`
- Debería ser: "su expresión"

**Línea 6**: `implicaría`
- Debería ser: "implicarían" (concordancia con "metales")

**Línea 16**: `múltiples ruidos blancos`
- Debería ser: "múltiples ruidos de fondo" o "ruido blanco" (singular)

**Línea 18**: `nexo-sistema`
- Inconsistente: en otros lugares se usa "Matrasy" sin guión
- **Sugerencia**: "sistema nexo" o "nexo del sistema"

**Línea 26**: `si no`
- Debería ser: "sino" (una sola palabra cuando significa "sino que")

**Línea 30**: `Mauro Tavira;`
- El punto y coma está mal usado, debería ser coma o punto

**Línea 35**: `preguntándole;`
- El punto y coma está mal usado, debería ser dos puntos o coma

**Línea 37**: `diciendo;`
- El punto y coma está mal usado, debería ser dos puntos o coma

**Línea 39**: `diciendo;`
- El punto y coma está mal usado, debería ser dos puntos o coma

**Línea 43**: `diciendo`
- Falta puntuación después de "diciendo"

**Línea 45**: `defenderse`
- Falta puntuación después de "defenderse"

**Línea 55**: `qué oculta`
- Debería ser: "que oculta" (sin tilde, es relativo, no interrogativo)

### 2. Repeticiones:

**Líneas 37, 39, 43, 45**: "diciendo" aparece múltiples veces
- Variar la estructura: "continúa", "agrega", "prosigue", etc.

**Línea 35**: "Viktor pregunta" y luego "Viktor pregunta" de nuevo
- Variar: "Viktor continúa", "Viktor insiste", etc.

### 3. Uso excesivo de punto y coma:

El punto y coma se usa incorrectamente en múltiples lugares. Debería usarse para:
- Separar elementos de una lista compleja
- Conectar oraciones relacionadas sin conjunción

No debería usarse para:
- Introducir diálogos (usar dos puntos)
- Separar sujeto y predicado (usar coma)
